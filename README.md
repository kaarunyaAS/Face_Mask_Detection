# 😷 Face Mask Detection System

A Deep Learning and Computer Vision project developed to detect whether a person is wearing a face mask or not using image classification techniques. This project was implemented using Python, OpenCV, TensorFlow/Keras, and Jupyter Notebook.

---

## 🚀 Project Overview

This project focuses on detecting face masks from images using Deep Learning techniques.  
The model is trained on face mask datasets containing masked and unmasked face images.

The project includes:
- Image preprocessing
- Dataset handling
- CNN/Deep Learning model training
- Prediction and evaluation
- Face mask classification

---

## ✨ Features

- 😷 Detects masked and unmasked faces
- 🧠 Deep Learning-based image classification
- 🖼️ Image preprocessing and augmentation
- 📊 Model training and evaluation
- 📈 Accuracy visualization
- 📓 Implemented in Jupyter Notebook
